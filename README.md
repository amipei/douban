# douban
学习React，制作的demo



## 页面入口

路由地址: /  =>(跳转)   /book or /movie or /music

请求地址:
https://api.douban.com/v2/book/search
https://api.douban.com/v2/music/search
https://api.douban.com/v2/movie/search
https://api.douban.com/v2/movie/top250

页面描述: 书籍/电影/音乐  列表信息

输入参数:  
| 名称 | 类型 | 描述 |
| ---  | --- | --- |
|  q  | String  | 搜索内容  |
| start | Number  | 开始位置  |
| count | Number  | 结果条数  |
